Commands

Mnemonic ID Description Arguments
Argument Type Description
CS_Run ['0'] Run a command sequence file
fileName Fw::CmdStringArg The name of the sequence file
CS_Validate ['1'] Validate a command sequence file
fileName Fw::CmdStringArg The name of the sequence file
CS_Cancel ['2'] Cancel a command sequence
CS_Start ['3'] Start running a command sequence
CS_Step ['4'] Perform one step in a command sequence. Valid only if CmdSequencer is in MANUAL run mode.
CS_Auto ['5'] Set the run mode to AUTO.
CS_Manual ['6'] Set the run mode to MANUAL.
